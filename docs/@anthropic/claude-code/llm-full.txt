# Claude Code Documentation

## Overview

Claude Code is Anthropic's agentic coding tool that lives in your terminal and helps you turn ideas into code faster than ever before.

### Get Started in 30 Seconds

**Prerequisites:** A Claude.ai or Claude Console account

**Install Claude Code:**

**macOS, Linux, WSL:**
```bash
curl -fsSL https://claude.ai/install.sh | bash
```

**Windows PowerShell:**
```powershell
irm https://claude.ai/install.ps1 | iex
```

**Windows CMD:**
```cmd
curl -fsSL https://claude.ai/install.cmd -o install.cmd && install.cmd && del install.cmd
```

**Homebrew:**
```bash
brew install --cask claude-code
```

**NPM (requires Node.js 18+):**
```bash
npm install -g @anthropic-ai/claude-code
```

**Start using Claude Code:**
```bash
cd your-project
claude
```

### What Claude Code Does

- **Build features from descriptions**: Tell Claude what you want to build in plain English
- **Debug and fix issues**: Describe bugs or paste error messages for analysis and fixes
- **Navigate any codebase**: Ask questions about your team's codebase
- **Automate tedious tasks**: Fix lint issues, resolve merge conflicts, write release notes

### Why Developers Love Claude Code

- **Works in your terminal**: Meets you where you already work
- **Takes action**: Directly edits files, runs commands, creates commits
- **Unix philosophy**: Composable and scriptable
- **Enterprise-ready**: API access, AWS/GCP hosting, enterprise security

## GitHub Actions Integration

### Basic Setup

```yaml
name: Claude Code Review
on:
  pull_request:
    types: [opened, synchronize]

jobs:
  claude-review:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - name: Claude Code Review
        uses: anthropics/claude-code-action@main
        with:
          api_key: ${{ secrets.CLAUDE_API_KEY }}
          prompt: "Review this PR for code quality and suggest improvements"
```

### Auto-fix Workflow

```yaml
name: Claude Auto-fix
on:
  push:
    branches: [main]

jobs:
  auto-fix:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - name: Fix linting issues
        uses: anthropics/claude-code-action@main
        with:
          api_key: ${{ secrets.CLAUDE_API_KEY }}
          prompt: "Fix all linting issues in the codebase"
```

## Model Context Protocol (MCP)

### What is MCP?

MCP allows Claude Code to connect to external data sources and tools.

### Configuration

**Add MCP servers to your config:**
```json
{
  "mcpServers": {
    "filesystem": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-filesystem", "/path/to/project"]
    },
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"],
      "env": {
        "GITHUB_TOKEN": "${GITHUB_TOKEN}"
      }
    }
  }
}
```

### Common MCP Servers

- **Filesystem**: Read/write files
- **GitHub**: Manage repositories, issues, PRs
- **Slack**: Send messages, read channels
- **Figma**: Access design files
- **Google Drive**: Read documents
- **Jira**: Manage tickets

## Configuration

### Settings Files

**Global settings:** `~/.claude/settings.json`

**Project settings:** `.claude/settings.json`

### Key Configuration Options

```json
{
  "model": "claude-3-5-sonnet-20241022",
  "max_tokens": 4096,
  "temperature": 0.1,
  "auto_update": true,
  "permissions": {
    "read": true,
    "write": true,
    "execute": false
  },
  "mcpServers": {},
  "hooks": {
    "pre_prompt": "echo 'Starting Claude session'",
    "post_response": "echo 'Response received'"
  }
}
```

## CLI Reference

### Basic Commands

```bash
# Start interactive session
claude

# Single prompt
claude "Your prompt here"

# Pipe input
echo "code" | claude -p "Analyze this"

# Read from file
claude -p "Review this code" < file.js

# Multiple prompts
claude -p "First task" -p "Second task"
```

### Command Options

```
claude [options] [prompt]

Options:
  -p, --prompt <prompt>     Single prompt mode
  -f, --file <file>         Read prompt from file
  -m, --model <model>       Specify model
  -t, --max-tokens <num>    Maximum tokens
  --temperature <temp>      Temperature setting
  --no-stream               Disable streaming
  --verbose                 Verbose output
  --debug                   Debug mode
  --version                 Show version
  --help                    Show help
```

### Interactive Mode Commands

**Slash commands:**
- `/help` - Show help
- `/clear` - Clear context
- `/exit` - Exit session
- `/commit` - Create git commit
- `/debug` - Toggle debug mode

## Advanced Features

### Subagents

Create specialized agents for specific tasks:

```json
{
  "subagents": {
    "frontend-expert": {
      "description": "React and TypeScript specialist",
      "model": "claude-3-5-sonnet",
      "skills": ["react", "typescript", "css"]
    },
    "backend-expert": {
      "description": "Python and database specialist",
      "model": "claude-3-5-sonnet",
      "skills": ["python", "sql", "api-design"]
    }
  }
}
```

### Hooks

**Pre and post execution hooks:**
```json
{
  "hooks": {
    "pre_prompt": ["npm run lint", "git status"],
    "post_response": ["npm test", "echo 'Done'"]
  }
}
```

## Troubleshooting

### Common Issues

**Installation fails:**
```bash
# Check Node.js version
node --version  # Should be 18+

# Clear npm cache
npm cache clean --force

# Try reinstalling
npm install -g @anthropic-ai/claude-code --force
```

**API errors:**
```bash
# Check API key
echo $CLAUDE_API_KEY

# Test connection
claude -p "Hello" --verbose
```

**Permission denied:**
```bash
# Check permissions
claude --debug

# Reset permissions
claude --reset-permissions
```
